Class: TreeManager service:gmfTreeManager

gmf.TreeManager

new TreeManager(gmfThemes)

Manage a tree with children. This service can be used in mode 'flush' or not (mode 'add'). In mode 'flush', each theme, group or group by layer that you add will replace the previous children's array. In mode 'add', children will be just pushed in this array. This service's theme is a GmfThemesNode with only children and a name. Thought to be the tree source of the gmf layertree directive.
